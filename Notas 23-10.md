## Estrutura de Dados

- Basicamente a estruturação de dados é útil para que a gente trabalhe com diferentes tipos de dados de forma que a gente precise resolver problemas de forma organizada e eficiente. Estruturas Lineares - Dados alinhados e Estruturas Não-Lineares são dados que podemos conseguir acessar independente da forma e conseguimos acessar

### Pilha

- Pilha é uma estrutura que trabalha com o LIFO (Last In, First Out) - O último dado a entrar é primeiro a sair. Ou seja, podemos realizar vários tipos de operações com os dados de forma que seja organizada respeitando a pilha.
