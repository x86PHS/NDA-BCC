## Notas de Aula
Este repositório mantém as notas de aula das disciplinas do Bacharelado em Ciências da Computação no Centro Universitário Senac.

## Objetivos
-   Centralizar todo o conteúdo das matérias em um só lugar;
-   Prover notas mais eficazes para as provas e consultas futuras.

## Créditos

As notas de aulas deste repositório foram construídas utilizando o template **Notas de Aulas**, desenvolvido por **Enzo Libório** (@enzoliborio).
