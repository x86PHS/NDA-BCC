 O SQL tem 3 tipos de agrupamento de comandos, de modo que há três tipos de categorias: DCL (Data Control Language), DDL (Data Definition Language) e DML (Data Manipulation Language).

DCL: Comandos para gerir/controlar usuários para acessar bancos de dados etc.
	- Grant
	- Revoke
	- Deny
DDL: Comandos para criar tabelas/bancos de dados/pk/índices
	- Create
	- Alter
	- Drop
DML: São comandos de manipulação para acessar dados, apagar etc.
	- Select
	- Insert
	- Update
	- Delete

Alguns SGBDS são case-sensitive. Diferente do Management Studio, nós podemos colocar vários tipos de comandos não case-sensitive, entretanto, há casos que é configurado para que seja rígido.
